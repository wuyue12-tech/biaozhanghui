# 八升九入境仪式成功标准

版本：v0.1  
状态：主线文档  
更新日期：2026-06-21

## 一句话标准

成功的八升九入境仪式，是让学生从“被安排进入九年级”，变成“参与设计、完成挑战、带着支持进入九年级”。

这份文档是后续策划的主线。活动形式、流程、道具、场地、家长参与、学生分工，都要回到这里校准。

## 当前主线

学生共创，团队闯关，家长见证，导师认证。

这条主线包含四个动作：

1. 学生在前期参与内容共创。
2. 学生在现场以导师小队为单位完成挑战。
3. 家长在现场看见孩子的团队状态，并完成支持动作。
4. 导师完成入境认证，让活动有清楚的收束。

## 成功对象

这场活动同时服务四类人：

| 对象 | 他们在活动中的身份 | 成功时应该发生什么 |
|---|---|---|
| 学生策划人 | 共创者 | 他们前期提供的真实内容进入活动现场 |
| 学生参与者 | 闯关者 | 他们在团队任务中完成动作、合作和选择 |
| 家长 | 见证人与支持者 | 他们看见九年级压力的真实样子，并写下一个具体支持动作 |
| 老师 | 组织者与认证者 | 他们能用清楚规则推进现场，并完成导师认证 |

## 学生作为策划人的成功标准

学生策划能力有限，所以前期共创采用“小任务 + 模板 + 样例 + 验收标准”的方式。

成功表现：

1. 活动里的九年级情境，至少一半来自学生前期收集。
2. 每个导师班完成一个局部策划任务。
3. 学生在现场拥有具体岗位。
4. 学生能看见自己的内容被采用。

可布置的学生前期任务：

| 任务 | 学生要交什么 | 进入活动的方式 |
|---|---|---|
| 收集九年级担心 | 每人写 1 个真实担心 | 情境卡 |
| 改写空话 | 把一句宽泛表达改成一个具体动作 | 行动卡 |
| 设计战队元素 | 队名、队标、入场动作 | 战队集结 |
| 采访家长 | 问家长最担心九年级哪件事 | 家长支持卡 |
| 试玩规则 | 体验 5 分钟，说出一个卡点 | 流程修正 |

学生共创内容进入现场时，尽量保留“来自八年级同学前期调研”的标识。

## 学生作为参与者的成功标准

现场成功，核心看学生是否进入状态。

成功表现：

1. 110 名学生中，至少 90 人有实际动作参与。
2. 每个导师小队完成一次团队协作任务。
3. 每队产出一张“九年级行动牌”。
4. 每张行动牌包含：真实挑战、具体动作、求助对象或检查方式。
5. 现场出现至少 3 个自然高点，例如入场、倒计时、闯关完成、导师认证、合影。

学生结束后能说清这句话的意思：

> 九年级遇到事，先把问题拆小，再找一个具体动作。

## 家长参与的成功标准

家长在活动中的定位是：入境见证人，也是九年级支持者。

成功表现：

1. 家长看见孩子在团队中行动，而不是只看见成绩、排名或表演。
2. 家长理解九年级压力来自作业、考试、睡眠、手机、关系、求助等具体处境。
3. 家长学会一种支持语言。
4. 家长现场写下一张“九年级支持卡”。

家长可以带走的支持语言：

| 场景 | 家长可以怎么问 |
|---|---|
| 孩子压力大 | 你这周最想先解决哪一件事？ |
| 孩子拖延 | 这件事能不能先拆成一个 20 分钟的小动作？ |
| 孩子考砸 | 你想先看哪一类错题？ |
| 孩子需要帮助 | 你需要我帮你创造什么条件？ |
| 亲子沟通紧张 | 我先听你说完，再一起看下一步。 |

家长支持卡只写一件可执行的小事。例如：

- 每周日晚上和孩子一起看一次任务安排。
- 考试后先听孩子说完，再一起看下一步。
- 晚上 10:30 后一起把手机放到客厅充电。
- 孩子需要问老师时，先帮他把问题写成一句话。

## 仪式感的成功标准

入境仪式需要有清楚的身份变化。

| 阶段 | 学生身份 | 现场动作 |
|---|---|---|
| 开场 | 八年级学生 | 入场、集结、领取任务 |
| 过程 | 入境挑战者 | 组队、闯关、协作、完成行动牌 |
| 结尾 | 九年级预备成员 | 导师认证、家长支持、入境完成 |

成功的仪式感来自真实完成，而不是长篇发言。现场需要有可见的完成物：任务板、行动牌、支持卡、认证章、合影。

## 组织落地的成功标准

活动规则要轻，现场推进要稳。

成功表现：

1. 主持人 3 分钟能讲清核心规则。
2. 学生 5 分钟内能进入任务。
3. 每队任务包一打开就知道下一步做什么。
4. 17 位老师都有明确位置和职责。
5. 室内篮球馆被有效使用：战队区、挑战区、导师认证区、家长观察区、入境完成区。

## 方案校准表

之后每个活动方案都要用这张表校准。

| 校准项 | 必须回答的问题 | 合格表现 |
|---|---|---|
| 学生共创 | 学生前期贡献了什么？ | 情境、行动、视觉、音乐、岗位中至少有 2 类来自学生 |
| 团队参与 | 团队为什么必须合作？ | 跑、选、贴、写、审、说等动作分散到不同学生 |
| 热血感 | 现场高点在哪里？ | 有倒计时、接力、对抗、完成、认证中的至少 3 个 |
| 教育意义 | 学生带走什么行动办法？ | 每队留下 1 张具体行动牌 |
| 家长体验 | 家长看到什么、做了什么？ | 看见孩子团队状态，并完成 1 张支持卡 |
| 老师执行 | 老师如何推进和判断？ | 老师拿到岗位卡和验收标准即可执行 |
| 场地适配 | 篮球馆如何被使用？ | 动线清楚，学生等待时间短，安全边界明确 |
| 收束动作 | 什么标志着入境完成？ | 导师认证、行动牌、家长支持卡形成闭环 |

## 方案准入线

一个方案进入细化阶段前，至少满足以下 5 条：

1. 110 名学生里，预计至少 90 人有实际动作参与。
2. 每个导师小队都有清楚分工，一个人包办会明显变慢。
3. 活动中至少有 3 个自然高点。
4. 每队能产出一条具体九年级行动。
5. 家长有清楚的观看点和参与点。

## 后续方案输出格式

之后每个版本的方案都应包含：

1. 方案名称。
2. 一句话主线。
3. 参与对象与角色。
4. 85 分钟流程。
5. 学生前期共创任务。
6. 家长参与设计。
7. 场地与物资。
8. 成功标准校准表。
9. 主要风险与调整办法。

## 当前保留判断

当前策划的主线暂定为：

> 学生共创，团队闯关，家长见证，导师认证。

活动形式可以继续比较篮球馆挑战赛、闯关赛、接力赛、任务站等版本。每个版本都用本文件校准后再进入细化。
